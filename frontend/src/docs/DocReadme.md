# 🎯 Frontend Setup Guide

Este documento descreve os passos necessários para configurar, instalar dependências e executar o projeto frontend localmente.

---

## 📌 Visão Geral

O frontend foi desenvolvido utilizando **React + Vite**, garantindo um ambiente moderno, rápido e eficiente para desenvolvimento.

---

## ⚙️ Pré-requisitos

Certifique-se de que possui as seguintes ferramentas instaladas no seu ambiente:

* **Node.js** (versão LTS recomendada)
* **npm** (gerenciador de pacotes)

Verifique as versões com:

```bash
node -v
npm -v
```

---

## 📥 Instalação do Projeto

Siga os passos abaixo para preparar o ambiente:

### 1. Clonar o repositório (caso ainda não tenha feito)

```bash
git clone <url-do-repositorio>
```

### 2. Acessar a pasta do frontend

```bash
cd frontend
```

### 3. Instalar as dependências

```bash
npm install
```

> Este comando irá instalar todas as dependências listadas no `package.json`.

---

## ▶️ Execução em Ambiente de Desenvolvimento

Para iniciar o servidor local:

```bash
npm run dev
```

Após executar, o terminal exibirá um endereço semelhante a:

```
http://localhost:5173
```

Abra este link no navegador para visualizar a aplicação.
